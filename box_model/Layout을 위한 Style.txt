- Block 요소 : 요소를 삽입했을 때 혼자 한줄을 차지하는 요소
  * 요소의 너비가 100%
  * ex) <div> <p> <h1> 등

- 인라인 요소 : 한줄을 차지하지 않는 요소
                즉, 화면에 표시되는 콘텐츠만큼만 영역을 차지함
  * ex) <img> <strong> <span> 등

- CSS BOX Model
  * Box Model 구성 : 
  * Box Model의 전체 너비 = width 값 + 좌우 패딩 + 좌우 테두리

- display 속성 : "Block 요소"를 "Inline 요소"로 바꾸거나
                 "Inline 요소"를 "Block 요소"로 바꾸는 것
  * display : block, inline, inline-blick
  

- position 속성 : 
  * position : static, relative, fixed, sticky, ...
